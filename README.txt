بوصلة | Boslah
مساعد الدراسة الذكي - المرحلة الثانوية

وصف المشروع:
تطبيق ويب مبتكر يهدف إلى تنظيم وقت الطالب بناءً على تحليل ذكي لنقاط الضعف والقوة في المواد الدراسية، ثم توليد جدول مذاكرة مناسب لمستوى الطالب.

--------------------------------------------------

التقنيات المستخدمة (Tech Stack)

- اللغات:
  HTML5 (تم دراستها بعمق في المرحلة الإعدادية)
  CSS3
  JavaScript (تم دراستها ضمن منهج المرحلة الثانوية)

- الوسائط المستخدمة:
  أيقونات مخصصة + خطوط ويب + صور رمزية (Favicons) لتحسين تجربة المستخدم والعرض البصري.

- التقنيات الحديثة:
  Progressive Web App (PWA)

- الاستضافة:
  GitHub Pages

--------------------------------------------------

طريقة التشغيل

1) العرض المباشر (Online)
يمكنك فتح الموقع فوراً عبر الرابط التالي:
https://jasseramir.github.io/boslah/

2) التشغيل محلياً (Local)
- قم بتحميل أو فك ضغط ملفات المشروع.
- افتح المجلد باستخدام VS Code.
- شغّل ملف index.html عن طريق إضافة Live Server.
(أو يمكنك فتح index.html مباشرة من المتصفح)

ملحوظة:
يعمل المشروع بدون إنترنت، وقد لا تظهر بعض الخطوط أو الأيقونات إذا لم تكن مخزنة محلياً.

--------------------------------------------------

فكرة الذكاء الاصطناعي في "بوصلة"

يعتمد المشروع على خوارزمية (Priority Algorithm) تقوم بـ:
1) تحليل درجات الطالب المدخلة.
2) تحديد المواد التي تحتاج إلى جهد أكبر.
3) توليد جدول مذاكرة ذكي يوزّع الوقت بكفاءة لتحقيق التعلم الشخصي.
