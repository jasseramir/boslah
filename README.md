## بوصلة | Boslah
### مساعد الدراسة الذكي - المرحلة الثانوية

> **وصف المشروع:** تطبيق ويب مبتكر يهدف إلى تنظيم وقت الطالب بناءً على تحليل ذكي لنقاط الضعف في المواد الدراسية وإنشاء جدول للمذاكرة.

---

## التقنيات المستخدمة (Tech Stack)
* **اللغات:** `HTML5` `CSS3` `JavaScript`
* **التقنيات الحديثة:** `PWA` (Progressive Web App)
* **الاستضافة:** `GitHub_Pages`
