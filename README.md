# بوصلة | Boslah
### مساعد الدراسة الذكي - المرحلة الثانوية

> **وصف المشروع:** تطبيق ويب مبتكر يهدف إلى تنظيم وقت الطالب بناءً على تحليل ذكي لنقاط الضعف والقوة في المواد الدراسية، ثم توليد جدول مذاكرة مناسب لمستوى الطالب.

---

## التقنيات المستخدمة (Tech Stack)

- **HTML5**  
  *(تم دراستها ضمن منهج المرحلة الإعدادية)*

- **CSS3**

- **JavaScript**  
  *(تم دراستها ضمن منهج المرحلة الثانوية)*

- **الوسائط المستخدمة:**  
  أيقونات مخصصة + خطوط ويب + صور رمزية (Favicons) لتحسين تجربة المستخدم والعرض البصري.

- **التقنيات الحديثة:**  
  Progressive Web App (PWA)

- **الاستضافة:**  
  GitHub Pages

---

## طريقة التشغيل

### 1) العرض المباشر (Online)
افتح الموقع عبر الرابط التالي:  
https://jasseramir.github.io/boslah/

### 2) التشغيل محليًا (Local)
- قم بتحميل أو فك ضغط ملفات المشروع.
- افتح المجلد باستخدام **VS Code**.
- شغّل ملف `index.html` عن طريق إضافة **Live Server**.  
  *(أو افتح `index.html` مباشرة من المتصفح)*

> **ملحوظة:** يعمل المشروع بدون إنترنت، وقد لا تظهر بعض الخطوط أو الأيقونات إذا لم تكن مخزنة محليًا.
